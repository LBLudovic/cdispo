
# Photos – remplacement manuel

Les images présentes dans ce ZIP sont **génériques (placeholders)**.

Pour une version plus réaliste, vous pouvez remplacer les fichiers
dans `assets/images/` par de **vraies photos** (libres de droit).

## Bonnes pratiques
- Résolution conseillée : **1200×800** ou plus
- Format : JPG ou WEBP
- Nommer les fichiers sans espaces (ex: `cocktail_1.jpg`)
- Conserver les mêmes noms pour éviter de modifier le code
